# SQL_PRACTISE

A structured collection of SQL practice queries for learning and reference.

## Structure

```
SQL_PRACTISE/
└── movies-db/
    ├── schema.sql   -- Table creation script
    ├── seed.sql     -- Sample data insertion
    └── queries.sql  -- SELECT query examples
```

## Getting Started

1. Run `schema.sql` to create the `movies` table
2. Run `seed.sql` to populate it with sample data
3. Run any query from `queries.sql` to practice

## Topics Covered

- `movies-db/` — Basic SELECT queries on a movies table
